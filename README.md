# AI Chatbot with Streamlit

This project implements a conversational AI chatbot with a web interface using Streamlit and Hugging Face Transformers.

The chatbot generates dynamic responses using a pre-trained language model.

## Features

* AI-powered conversation
* Interactive web interface
* Chat history support

## Technologies Used

Python
Streamlit
Hugging Face Transformers

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the chatbot:

streamlit run app.py

Then open the browser link shown in the terminal.
