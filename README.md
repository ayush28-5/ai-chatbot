# AI Chatbot with Streamlit

This project implements a conversational **AI chatbot with a web interface** using **Streamlit** and **Hugging Face Transformers**.

The chatbot generates dynamic responses using a pre-trained language model and provides a simple chat interface similar to modern AI chat applications.

---

## Features

* AI-powered conversation
* Interactive web interface
* Chat history support
* Real-time response generation

---

## Technologies Used

* Python
* Streamlit
* Hugging Face Transformers
* PyTorch

## How to Run

### 1. Install dependencies

pip install -r requirements.txt

### 2. Run the chatbot

streamlit run app.py

### 3. Open the browser

After running the command, Streamlit will display a local URL in the terminal (usually http://localhost:8501).
Open that link in your browser to interact with the chatbot.
